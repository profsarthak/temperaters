# temperaters

Documentation and code for an **India temperature geospatial mapping** project by
**Shreyansh Dwivedy** and **Sarthak Mohanty**.

The project explores spatial patterns in temperature across India — combining temperature
records with geographic data to map and analyse how temperatures vary across regions.

## Status

Early stage. This repository currently serves as the project's home; code, data
documentation, and outputs will be added here as the work develops.
